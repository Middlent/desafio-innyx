<h1>Instalação</h1>
Após clonar o repositório entre na pasta desafio-back-end e faça uma cópia do arquivo .env.example chamada .env
Dentro do .env altere as opções que começam com DB_ para se adequar ao seu banco de dados.

Após criado o .env execute o setup.bat e espere o cmd fechar.

<h1>Execução</h1>
Execute ambos o run_server.bat e o run_app.bat.

Acesse localhost:5173 para acessar o app.

<h1>Login</h1>
Na primeira vez que acessar o app será criado um perfil com login admin@adm e senha admin.
Esse perfil tem super adm e deve ser usado para dar acesso a quaisquer outro perfil cadastrado.
